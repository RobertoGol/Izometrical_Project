# Audio Map for Izometrical Project

This document lists all generated audio files (ogg) and their intended use for maximum player immersion.

## 1. Robot ROY (Your Main Sidekick)
- `roy_online.ogg`: Initial boot-up sequence.
- `roy_target.ogg`: Combat start.
- `roy_critical.ogg`: Low HP warning.
- `roy_scanning.ogg`: Exploration mode.
- `roy_welcome_ru.ogg`: Interaction start.
- `roy_ammo_low.ogg`: Inventory alert.
- `roy_repair_ru_v2.ogg`: Healing initiated.

## 2. Combat & Action (Multilingual Slang)
- `rus_combat_v3.ogg`, `rus_aggressive_push.ogg`: High-intensity Russian combat shouts.
- `rus_reload_profane.ogg`: Tactical reloading (Russian).
- `wpn_shot_ja.ogg`, `magic_mega_ja.ogg`: Anime-style combat and ultimate moves.
- `wpn_talk_sniper.ogg`: Weapon OS (Sniper rifle).
- `wpn_plasma_de.ogg`: Weapon OS (Energy weapons).

## 3. Magic & Special Abilities (High Level)
- `magic_final_judgment.ogg`: Celestial/Light magic.
- `magic_dark_de.ogg`: Dark/Forbidden magic.
- `magic_ultimate_ja.ogg`: Absolute Zero/Anime finisher.
- `rus_summon_wow.ogg`: Casual summon reaction (Russian).

## 4. World, Nature & Radio
- `radio_host_bunker.ogg`: Main Bunker Radio DJ.
- `radio_quest_hook_ru.ogg`: Quest-giver radio broadcast.
- `env_swamp_ru.ogg`, `env_mountain_en.ogg`: Location descriptors.
- `quest_sos_signal.ogg`: Dynamic world event (Rescue mission).
- `anom_glitch_v2.ogg`: Bunker anomaly effect.

## 5. Musical Elements (Vocal Tracks)
- `music_boss_latin.ogg`: Layer this over epic drums for Boss fights.
- `music_anime_hype.ogg`: Use for combat-ready state or power-ups.
- `music_radio_jingle.ogg`: Radio station ID.
- `music_cult_chant.ogg`: Dark cult/Bunker background music.

*Note: For instrumental music (drums, synths, strings), refer to OpenGameArt.org or Incompetech.*
